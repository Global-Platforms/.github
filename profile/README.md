### Technology Landscape

We operate a cloud-native stack heavily leveraging **Azure**, **Kubernetes**, and **GitHub Actions** for automation.

| Domain | Technologies |
|:---|:---|
| **Computation** | Azure Kubernetes Service (AKS), Azure Container Apps, Serverless Functions |
| **IaC & Config** | Terraform, Helm, Ansible, YAML Pipelines |
| **Automation** | GitHub Actions (Runners & Composite Actions), Python Scripting |
| **Compliance** | Automated Policy Enforcement, Audit Log Retention, Secret Scanning |

---

### Engineering Principles

- **Everything as Code** — Infrastructure, configuration, and policies are版本 managed in Git.
- **Fail Fast, Fix Faster** — Automated testing and rapid feedback loops in CI/CD.
- **Security by Design** — Least privilege access, automated secret rotation, and strict network policies.
- **Golden Paths** — Providing standardized, secure, and easy-to-use templates for developers.
