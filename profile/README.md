### About

**Global Platforms** is the platform engineering team at Telefonica Germany. We build and operate the internal developer platform that powers identity management, API operations, deployment automation, compliance tooling, and quality assurance across all brands and environments.

```
Platform Engineering  ·  Cloud Infrastructure  ·  Automation  ·  Reliability
```

---

### What We Do

<table>
<tr>
<td width="50%" valign="top">

<h4>Identity & API Operations</h4>

- OAuth2 / OIDC automation across multi-tenant environments
- API lifecycle management and gateway configuration
- Cross-brand credential and session management
- Purpose and consent orchestration

<h4>Automation & Tooling</h4>

- CLI toolkits for bulk API and tenant operations
- Data processing pipelines and scheduled reporting
- Migration tooling across environments and tenants
- Self-service onboarding for product teams

</td>
<td width="50%" valign="top">

<h4>DevOps & Infrastructure</h4>

- CI/CD orchestration with GitHub Actions
- Multi-service deployment across environments
- Azure Kubernetes Service (AKS) operations
- Infrastructure-as-Code with Terraform

<h4>Compliance & Reliability</h4>

- GDPR-compliant audit dataset lifecycle management
- Billing and cost analysis automation
- BDD-driven end-to-end API testing
- Environment monitoring and incident response

</td>
</tr>
</table>

### Engineering Principles

<table>
<tr>
<td>

- **Everything as Code** &mdash; Infrastructure, configuration, and policies are version-controlled in Git
- **Fail Fast, Fix Faster** &mdash; Automated testing and rapid feedback loops in CI/CD
- **Security by Design** &mdash; Least privilege access, automated secret rotation, and strict network policies
- **Golden Paths** &mdash; Standardized, secure, and easy-to-use templates for product teams
